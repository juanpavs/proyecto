Si no esta generando conexión a la placa esp32, aqui se puede seguir una serie de pasos para solucionar ese problema:
1. Entrar al cmd y colocar el comando "ipconfig" donde se verifican los datos de la dirección ipv4, donde se opera la conexión del equipo y la base de datos, y las otras mascaras de red, donde se mostra la conexión del equipo.
2. Una vez que se halla encontrado la direccion ip, modifique los datos numericos, dependiendo de la red en la linea y modifique este comando:
IPAddress server_addr(192,168,0,100);
a la dirección obtenida por el cmd.
3. Modifique las lineas de las red de ejemplo a la red conectada y su contraseña respectiva en los metodos de ssid y password usados:
const char* ssid = "dlink";
const char* wifipassword = "";
donde se modifica las lineas entre comillas, para poder acceder a la red wifi en el codigo de la carpeta 2 en el sketch_nov24b.
4. Instale los programas XAMPP y Mysql Workbench si no los tiene instalados, si los tiene instalados, omita este paso y siga con el paso 5.
5. Configure en mysql workbench, las credenciales del codigo de react, es decir los parametros de conexión en index.js, de la carpeta server y agregelos en mysql workbench y las carpetas de configuración y el cmd de xampp, empleandose tal cual, estan declaradas.
6. Crre la conexión del proyecto con el nombre de proyecto, con el usuario de nombre: root y la contraseña y los credenciales previamente mencionados y las rutas de los mismos dichos anteriormente ahi en el paso 5.
7. Cree en la seccion de privilegios de phpmyadmin, al darle click al boton de admin en Xampp, en el cual cree el usuario: "proyecto" con la contraseña igual, es decir "proyecto" tambien y que tengan acceso desde cualquier servidor, agregando esta linea para ello: %.%.%.% y concediendo todos los privilegios y accesos desde phpmyadmin para las ip generica de la placa esp32 a la red wifi y la base de datos empleada.
8. Agrege estas lineas en el cmd de xampp, las credenciales del usuario y contraseña para el acceso desde la carpeta en txt del xampp en el disco local c, modifique las credenciales como se menciono en los pasos anteriores y adicione esas credenciales y modifique el mariadb.
9. Ingrese al cmd de xampp con las credenciales del usuario con privilegios del paso 6 y verifique su acceso para poder buscar la ip de mariadb y verificar si esta conectada a la red y unida a la conexión del sistema, si no esta vaya al archivo de la carpeta y modifique el archivo para el acceso de las credenciales al servicio y verifique si la ip empleada de mariadb es la misma o no y que mantenga conexión continua, sea en el sitio empleado de la conexión o en otro, mostrando que si tiene acceso a la base de datos.
10. Ejecute los comandos del enviar en arduino, npm start y node index.js en dos terminales de react con la ruta a la carpeta server de este ultimo, por independiente, y con Xampp tambien que este en ejecución y comprobar las otras direcciones, donde se muestra que esta en servicio el sistema para las diferentes y empleando los elementos del circuito (la placa esp32 y sensor ultrasonico) para ensayo del mismo y puedan funcionar como se observa en el documento anexo del enlace que contiene este txt.
11. Si no funciona, revise alguna falla desde el paso 1 o otro para comprobar alguna falla y descartar algun caso empleado y verificar si es algun comando faltante o linea pendiente o lectura del equipo que emplea los programas o reiniciar estos mismos o el equipo completo, si persiste la dificultad y reencender estos programas del paso 10 y hacerlo de nuevo y esperar a las conexiones y funciones de esos casos dados.
